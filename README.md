# SpaceWar Game

## Overview
This project is a desktop-based space shooter game developed using C#. The player controls a spaceship, avoids obstacles, and fights enemies.

## Technologies
- C#
- .NET

## Features
- Player movement and shooting
- Enemy spawning
- Collision detection
- Score system

## Run
Open the project in Visual Studio and run the application.
